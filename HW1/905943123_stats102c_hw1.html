Problem 5: Let \(X \sim\) two-parameter \(Exponential(\lambda, \eta)\) with probability density function \[f(x) = \lambda e-\lambda(x-\eta), x \ge \eta\] where \(\lambda\) and \(\eta\) are positive constant.

  1. Find the cumulative density function F(x)

\[F(x) = 1-e^{-\lambda(x-\eta)}\]

  1. Write an algorithm using the CDF inverse method to generate random numbers from Exponential\((\lambda, \eta)\)

\[u = 1-e^{-\lambda(x-\eta)}\] \[ln(1-u) = -\lambda(x-\eta)\] \[\frac{ln(1-u)}{\lambda} = \eta-x\] \[\eta - \frac{ln(1-u)}{\lambda} = x\]

sample_exp <- function(lambda, eta, n){
  
  u <- runif(n)
  return(eta - ((log(1-u))/lambda))
}

Problem 6: The Pareto distribution function has a probability density as shown below with the shape parameter \(\alpha\) and the scale parameter \(\beta\).

\[\frac{\alpha\beta^\alpha}{x^{\alpha+1}}\] \[x > \beta\] (a) Find the cumulative density function F(x)

\[F(x) = \int^x_\beta\frac{\alpha\beta^\alpha}{t^{\alpha+1}} dt = 1-(\frac{\beta}{x})^\alpha\]

  1. Write an algorithm using the CDF inverse method to generate random numbers from \(Pareto(\alpha = 5, \beta = 1)\) \[u = 1-(\frac{\beta}{x})^\alpha\] \[1-u = (\frac{\beta}{x})^\alpha\] \[\frac{log(1-u)}{\alpha} = log(\beta)-\log(x)\] \[log(\beta) - \frac{log(1-u)}{\alpha} = log(x)\] \[x = e^{log(\beta) - \frac{log(1-u)}{\alpha}}\]
Pareto <- function(n){
  
  u <- runif(n)
  return(exp(log(1) - (log(1-u)/5)))
  
}
